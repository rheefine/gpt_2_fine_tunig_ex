# GPT 2 Model fine-tuning example

## Overview
GPT-2 모델을 사용한 간단한 파인튜닝 예제입니다

## How To

### Setup
```console
$ git clone https://github.com/rheefine/gpt_2_fine_tunig_ex.git
$ conda env create -f environment.yml
```

### Dataset
`data/data_ex.json`을 참고하여 해당 형식에 맞춰 데이터셋를 추가합니다. 

### Run
```console
$ python3 src/main.py
```
